# Plugin GEE GPR Phenology para QGIS

Este directorio está reservado para incorporar el plugin **GEE GPR Phenology** como submódulo o copia local del repositorio:

https://github.com/jf-floresriera/GEE_GPR_Phenology

En el informe final, el plugin se presenta como un resultado instrumental complementario al flujo principal implementado en Google Earth Engine y Python. No reemplaza los scripts usados para generar los resultados del manuscrito, sino que documenta la transferencia del flujo a un entorno QGIS 3.
